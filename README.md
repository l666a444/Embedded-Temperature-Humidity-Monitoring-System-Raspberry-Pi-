# Embedded-Temperature-Humidity-Monitoring-System-Raspberry-Pi-
Embedded system using Raspberry Pi and sensors for real-time environmental monitoring


## Overview
This project implements an embedded monitoring system using Raspberry Pi and environmental sensors to track temperature and humidity in real time. The system supports wireless communication and alert mechanisms for abnormal conditions.

## Features
- Real-time temperature and humidity monitoring  
- Bluetooth-based data transmission  
- Image capture capability for remote observation  
- Threshold-based alert system for abnormal conditions

## System Architecture
- Sensor data acquisition (temperature & humidity)
- Data processing on Raspberry Pi
- Wireless transmission via Bluetooth
- Alert triggering based on predefined thresholds

## Tools & Technologies
- Raspberry Pi  
- Python  
- Environmental sensors (temperature & humidity)  
- Bluetooth module  

## Results
The system successfully monitors environmental conditions in real time and triggers alerts when values exceed predefined thresholds, enabling remote monitoring and quick response.

This project demonstrates embedded system integration, sensor interfacing, and basic wireless communication for real-world monitoring applications.
